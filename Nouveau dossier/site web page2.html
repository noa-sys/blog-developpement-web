Les différentes étapes pour écrire sa première page web ou créer son premier site web :

Dans cette partie du blog nous allons parler des différentes étapes pour réaliser son projet web :


1ère étape:





Réflechir
penser

Pour bien commencer son projet il est indispensable
de savoir d'avance ce que l'on veut faire




2ème étape :





Modéliser
Ecrire

Une fois que vous avez réflchi a ce que vous voulez faire,
prenez une feuille et dessiner le plus précisement possible votre idée et une fois OK sur votre dessin,
passer a l'étape suivante.




3ème étape :





Installer un éditeur de texte

Pour écrire votre site web il faudra installer un éditeur de texte (vous pouvez écrire votre code dans bloc note )mais le mieux c'est :

Sublime text!




4ème étape :





Ecrire son code
vérifier la syntaxe

Pour créer son site web, la meilleure étape et seul de l'écriture
du code et de la verification de la syntaxe




Dernière étape :




styliser son site
mettre en ligne

Et oui! avant de finir son site, il faut le styliser
c'est a dire joindre une feuille "css" pour
donner de la couleur a votre site et puis trouver un hébergeur dans la liste présente dans la page précedente.

Comme vous pouvez le constater... Construire son site ou blog demande énormement de travail et de réflexion, et ça ne se fait pas en un claquement de doigt!
